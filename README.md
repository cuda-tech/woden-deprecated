### 技术选型
* 后端：Spring(Kotlin) + MySQL
* 前端：VUE
* 构建工具：gradle

### 代码结构
```
.
├── annotation-processor    # 注解处理器
├── frontend                # 前端工程
├── scheduler               # 调度模块
├── service                 # 数据服务层模块
└── webserver               # 管理系统
```
